Estructuras condicionales if-else y switch

Con este script lo que se consigue es crear una alerta reactiva que en función de lo que escribas te dé un mensaje u otro. Para ello se utiliza la etiqueta if-else y switch. Con la primera se consigue que al poner algo, te redirija a otro, y else se utiliza para que si no se ha puesto nada te salga otra alerta. El switch es una expresión que compara el valor de esa expresión para ejecutar declaraciones asociadas a ese case
